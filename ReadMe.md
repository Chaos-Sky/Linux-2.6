# Linux 2.6 内核教程

## 文件目录结构 

    / 所有的md文件
    /img 相关图片
    /src 资源文件
        /src/driver 驱动文件
        /src/kernel 内核/根文件系统等
        /src/software 软件(三环)
        /src/tools  windows下的内核源码阅读工具
        /src/pdf 一些参考数据和资料
        /src/script 实验用的脚本
    /test 一些笔记，未整理
